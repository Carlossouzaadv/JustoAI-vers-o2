## Contingencies & Pivots

### If Market Adoption Slow
- **Action:** Pivot to in-house legal (B2B2C)
- **Timeline:** Month 6 decision point
- **Impact:** Different GTM strategy

### If JUDIT Changes
- **Action:** Build alternative data sources
- **Timeline:** Month 3 research
- **Impact:** Reduces moat dependency

### If Competitors Enter
- **Action:** Accelerate feature development
- **Timeline:** Continuous
- **Impact:** Increase switching cost via features

### If AI Models Improve
- **Action:** Upgrade Gemini integration
- **Timeline:** Quarterly
- **Impact:** Better analysis, better retention

## Customer Involvement

**How we involve customers in roadmap:**

1. **Feedback loops:**
   - Monthly user survey (what features they want)
   - Support tickets (what problems they face)
   - NPS comments (why they'd/wouldn't recommend)
   - Feature requests (voting in public roadmap)

2. **Advisory board (Quarterly):**
   - 5–10 key customers
   - Deep feedback sessions
   - Early feature access
   - Influence roadmap priorities

3. **Public roadmap:**
   - Customers see what's coming
   - Vote on features (most requested first)
   - Transparency builds trust
   - GitHub projects (public tracking)
